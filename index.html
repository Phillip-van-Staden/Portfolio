Evently - Event Management Platform

Evently is an intuitive and powerful platform designed to streamline event management for both organizers and attendees. It offers secure login and role-based access through Clerk Authentication, allowing event organizers to easily create, edit, delete, and view events with rich details such as date, time, location, and images. The platform features seamless image uploads via UploadThing, advanced event search and filtering, and related event suggestions to enhance user experience. With secure payment processing through Stripe, users can effortlessly purchase tickets and manage their order history. Event organizers gain valuable insights through analytics, optimizing future events with data on ticket sales, and engagement.

Technology Stack: Next.js, Node.js, MongoDB, TypeScript, Tailwind CSS, Uploadthing, Clerk and Stripe
